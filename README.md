# 📊 Marketing Campaign Analytics Dashboard

## 🚀 Project Overview

This project is an end-to-end **Data Analytics solution** built using **PostgreSQL, SQL, and Power BI**.

It transforms raw marketing campaign data into a **star schema data warehouse** and provides interactive dashboards for performance analysis and decision-making.

---

## 🎯 Objective

To analyze marketing campaign performance and answer:

* Which channel gives highest ROI?
* Which campaigns have high acquisition cost?
* What are the monthly performance trends?
* Where should budget be optimized?

---
